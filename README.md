Sistema de Gestion basica de Inventario (Python + JSON)

Este es un sitema de consola interactiva para gestionar productos de una tiena basica, Utilizando archivos JSON para gestionar la informacion

Funcionalidades:
--Agragar un nuevo producto: registyra nombre, precio y si hay disponible en stock
--Ver el inventario: podemo ver todso los producto que hay en el archivo JSON
--Actualizar u producto: podemos cambiar el nombre, precio y la disponibilidad del producto
--Borrar producto: borra el producto mediante al nombre

Tecnología:
--Python 3
--JSON (Persistencia de Datos)
--Git/Github (Control de versiones)
